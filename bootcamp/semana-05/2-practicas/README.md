# Prácticas - Semana 05: Gestión de Estado

Contenido y ejercicios prácticos para reforzar conceptos de estado.

| #   | Práctica             | Descripción                                     | Tiempo estimado |
| --- | -------------------- | ----------------------------------------------- | --------------- |
| 01  | estado-local         | Contador y lista con `setState()`               | 30 min          |
| 02  | provider-cart        | Carrito usando `Provider` (`ChangeNotifier`)    | 60 min          |
| 03  | selector-performance | Optimización con `Selector` y `select`          | 45 min          |
| 04  | bloc-cubit           | Implementar `Cart` con `Cubit` (`flutter_bloc`) | 60 min          |
| 05  | riverpod-advanced    | Reescribir carrito con `Riverpod` + tests       | 75 min          |

---

Cada práctica debe incluir:

- Descripción del problema
- Requerimientos mínimos
- Sugerencias de organización de carpetas
- Criterios de evaluación (funcionalidad, código, documentación)
