# Práctica 01: Estado Local con `setState()`

## Descripción

Crear una pequeña app donde el estado se maneje con `StatefulWidget` y `setState()`:

- Pantalla principal con una lista de tareas.
- Botón para añadir tarea (modal con TextField).
- Cada tarea puede marcarse como completada.

## Requisitos mínimos

- Uso correcto de `setState()` para actualizar la lista.
- Mantener el scroll y estado al añadir elementos.
- `initState()` y `dispose()` usados donde corresponda.

## Criterios de evaluación

- Funcionalidad (3 pts)
- Código limpio (1.5 pts)
- Documentación y comentarios (0.5 pts)
