# Práctica 04: Implementación con BLoC / Cubit

## Descripción

Implementar la lógica del carrito usando `flutter_bloc` y `Cubit`:

- `CartCubit` con estados básicos y métodos `add`/`remove`.
- UI usando `BlocProvider` y `BlocBuilder`.
- Pruebas unitarias para `CartCubit`.

## Requisitos mínimos

- `CartCubit` con `emit` de estados correctos.
- Uso de `BlocBuilder` para renderizar la UI.
- Tests básicos (usar `bloc_test`).

## Criterios de evaluación

- Implementación funcional (3 pts)
- Tests mínimos (2 pts)
- Código y documentación (1 pt)
